```json
        },

    },

    "is_valid": true,

    "metadata": {

    }
}
```
</format>

<clarification>
instruction - остается всегда пустым
input - ввод пользователя для генерации ситуации. Должен вводиться вместе с ситуацией
output - полная конфликтная ситуация
conflict_info - информация о конфликте
situation - только текст ситуации
conflict_type - типо конфликта: Семейный/Деловой/Дружественный
attack/defence - агрессор/защита соответственно
name - персонаж в ситуации
formal_role - формальная роль в кейсе
is_public - буличный ли это конфликт
negotiation_subject - тема на переговоры
attack_trigger - причина, из-за которой агрессор начинает говорить
context - что окружает учасников конфликта
metadata - доп информация
industry - сфера, в которой происходит конфликт
</clarification>

</output>
